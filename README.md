# forum-front-end-vue
It's a forum for restaurants, and it's built by Vue framework.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files

```
npm run lint
```
